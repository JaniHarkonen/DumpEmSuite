git branch - näyttää branchit

git branch x - luo branchin x

git checkout x - vaihtaa branchiin x

git add .

git commit -m "kommentti"

git push -u origin x - puskee muutokset branchiin x

git pull origin master - pullaa masterin

git branch -D x - poistaa branchin x

git remote show origin - näyttää repon, johon lokaali repo on yhdistetty

git remote remove origin - poistaa remote repon asetuksista (ei siis poista repoa)

git remote add origin x - lisää remote repon asetuksiin (x on linkki repoon)

git tag -a x -m y  - luo ANNOTOIDUN tagin x viestillä y, HUOM viesti on PAKOLLINEN

git push origin x  - puskee muutokset tagiin x


* PERSONAL ACCESS TOKENIN KÄYTTÖ *
- poistetaan remote repo asetuksista:
	git remote remove origin

- lisätään remote repo PATia käyttäen:
	git remote add origin https://<token>@<git_url>.git

- pullataan repo PATia käyttäen:
	git pull https://<token>@<git_url>.git
